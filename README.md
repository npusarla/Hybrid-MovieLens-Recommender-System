# Hybrid-MovieLens-Recommender-System
Hybrid Recommender that uses a content based model, matrix factorization, and a deep learning model to provide top movie recommendations for users.

## Repository Contents 
src folder:
- Final Project.ipynb: Final Project Notebook with report embedded
- DLExploration.ipynb: Model exploration on deep learning model
- Item Based Collaborative Filtering and MF Exploration.ipynb: contains model exploration on the item based collaborative filtering baseline model we used
- Other_Baseline_Content_Based.ipynb: Alternative content based model implementation and baseline implementation(was considered for final project)
- best_model.h5: optimized deep learning model
